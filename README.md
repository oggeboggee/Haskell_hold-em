## Introduction

This repository represents our implementation of the card game Texas Hold'em made for the course Functional Prorgramming Project (DIT216).

The project contains a Texas Hold'em engine with modules for: evaluating hands, defining rules and actions, an eventbased system, all applied using stateful computations. In addition to this there is a module containing the logic and implementations needed in order to run the game on a localhost in order to play the game in multiple clients (browser tabs). The test folder contains ready-made tests consisting of unit tests in the form of test trees. The featured tests test both the hand evaluating component along with selective tests on the stateful computations.
